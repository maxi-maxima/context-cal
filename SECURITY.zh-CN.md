# 安全说明

`context-cal` 是本地优先工具，不调用模型 API、远程服务或遥测端点。

## 报告漏洞

请通过 GitHub security advisory，或 npm 包维护者邮箱报告漏洞。

请包含：

- 受影响版本
- 操作系统
- 可复现问题的命令
- 是否需要不可信仓库内容才能触发

## 范围

范围内：

- 通过报告生成或解析导致的任意代码执行
- 输出写入中的路径穿越
- 对仓库文件的不安全处理

范围外：

- 关于演示内容中刻意伪造数据的报告
- secret scanning 或 prompt-injection 检测请求
- 需要调用实时模型才能确认的问题
